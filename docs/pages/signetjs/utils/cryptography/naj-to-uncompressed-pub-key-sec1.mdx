# najToUncompressedPubKeySEC1

The `najToUncompressedPubKeySEC1` method converts a NAJ (NEAR Account JSON) public key to an uncompressed SEC1 public key format. This is useful when working with NEAR keys in Ethereum or other blockchain contexts.

## Usage

```ts twoslash
import { utils } from 'signet.js'

const najPublicKey = 'secp256k1:3Ww8iF...'
const sec1Key = utils.cryptography.najToUncompressedPubKeySEC1(najPublicKey)
```

## Parameters

| Parameter      | Type           | Description                                                                                                                                |
| -------------- | -------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `najPublicKey` | `NajPublicKey` | The NAJ public key to convert (e.g., `secp256k1:3Ww8iFjqTHufye5aRGUvrQqETegR4gVUcW8FX5xzscaN9ENhpkffojsxJwi6N1RbbHMTxYa9UyKeqK3fsMuwxjR5`) |

## Returns

| Type                     | Description                                 |
| ------------------------ | ------------------------------------------- | --- | --- | --- | --- |
| `UncompressedPubKeySEC1` | The uncompressed SEC1 public key (e.g., `04 |     | x   |     | y`) |
